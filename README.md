# money-trace
MoneyTrace（お金/跡,痕跡）, シンプルなお金管理アプリ

![Balance Page](https://github.com/sanflat/cabinet/blob/main/images/balancePage.png "Balance Page")

# backend & db
```
cd money-trace  
docker-compose build  
docker-compose up
```
# frontend
```
cd money-trace/frontend/  
pnpm install  
pnpm dev
```
